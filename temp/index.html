Optimizing Crop Recommendations Using NPK Values and Machine Learning Models

×

This project focuses on developing a hybrid machine learning model to provide accurate crop recommendations based on soil nutrients (Nitrogen, Phosphorus, Potassium) and environmental factors like temperature, humidity, pH, and rainfall. By combining Linear Regression and Random Forest, the model achieves a high accuracy of 99.3%, effectively capturing both linear and non-linear relationships in the data. The system aims to assist farmers in making data-driven decisions, optimizing crop yield, and improving resource utilization. This work demonstrates the potential of machine learning in precision agriculture, paving the way for scalable and real-time farming solutions.
